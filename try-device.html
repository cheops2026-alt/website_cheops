Capture with Your Camera

Take a photo of any object. The image is processed locally — nothing is uploaded.

Your browser will ask for camera permission.

Position the object clearly in the frame, then press Capture.

Download Photo

Photo captured! Download it, then use it with runcv.py as shown below.

Run runcv.py on Your Device

Follow these steps to generate a 3D model from the photo you just captured.

1

Clone the Repository

git clone https://github.com/cheops2026-alt/Photogrammetric-3D-Model-
View on GitHub
2

Install Requirements

Python ≥ 3.8 required. Install CUDA if available.

pip install -r requirements.txt pip install opencv-python
3

Run with Your Photo

Replace photo.png with your downloaded filename.

python runcv.py cheops_capture.png --output-dir output/
4

Or Use Live Webcam

Opens your camera directly. Press SPACE or S to capture.

python runcv.py

Key Options

Option Description
--captureExplicitly use webcam capture mode
--camera NUse camera device ID (default: 0)
--no-bake-textureUse vertex colors — faster generation
--no-viewerSkip opening the 3D viewer after generation
--output-dir path/Where to save the 3D model
